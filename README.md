# tg-group

这个仓库现在有两种方案：

## 方案 A：API 直发（旧）
- 文件：`scheduler.py`
- 作用：直接用账号会话/API 定时发消息
- 说明：这不是“模拟手动设置”

## 方案 B：Telegram Desktop 模拟手动设置（你现在要的）
- 文件：`desktop_scheduler_gui.py`
- 双击启动：`run_gui.bat`
- 作用：
  - 你先登录 Telegram Desktop 账号
  - 打开目标群聊
  - 这个工具帮你批量点击：
    - 右键发送按钮
    - 选择“定时发送”
    - 选择“重复：每天”
    - 把时间设置成 `00:00 / 00:10 / 00:20 ...`
  - 本质上就是**模拟你手动一个个设置**

---

## 安装依赖

```bash
pip install -r requirements.txt
```

## 启动 GUI

### 方式 1：命令行
```bash
python desktop_scheduler_gui.py
```

### 方式 2：双击
直接双击：
```bash
run_gui.bat
```

---

## 使用步骤（重点）

### 1. 先打开 Telegram Desktop，并登录你的账号
这个工具**不是机器人发消息**。
它是操作你已经登录好的 Telegram Desktop。

### 2. 打开你要设置的那个群聊
先手动进入群聊聊天窗口。

### 3. 在 GUI 里填写：
- 开始时间：比如 `00:00`
- 结束时间：比如 `23:50`
- 间隔：`10`
- 文案池：你要发送的内容
- 文案模式：
  - `rotate` = 轮播
  - `random` = 随机

### 4. 记录坐标
把鼠标移动到对应位置，再点 GUI 里的“记录当前位置”：
- 输入框
- 发送按钮
- 右键菜单里的“定时发送”
- 定时弹窗里的时间输入框
- “重复”下拉框
- “每天”选项
- 最后的“定时”按钮

### 5. 开始批量设置
点“开始批量设置”即可。

工具会帮你批量创建：
- 00:00 每天
- 00:10 每天
- 00:20 每天
- ...
- 23:50 每天

---

## 适合你的场景
如果 Telegram Desktop 本身只能一次设置一个“每天重复”，
那这个工具就等于替你把这些动作连续做完。

---

## 注意事项
1. 运行时不要乱动鼠标键盘。
2. 如果出错，把鼠标快速移动到屏幕左上角，会触发 PyAutoGUI 的紧急停止。
3. Telegram 界面变了、缩放变了、语言变了，都可能需要重新记录坐标。
4. 建议先在测试群里试几条，确认没问题再一次性全跑。
5. 这是桌面自动化，不是 Telegram 官方原生批量重复接口。

---

## 你现在真正要的点
你说的“不是工具发，是帮我设置 Telegram 里的定时发送”，
这个 GUI 就是按这个思路做的：

**它不是代发，而是帮你在 Telegram Desktop 里批量点“定时发送 / 重复每天”。**
