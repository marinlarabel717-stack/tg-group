# tg-group

这个仓库当前包含两部分：

## 1. UI 原型软件（当前推荐先看这个）
- 文件：`ui_prototype_v1.py`
- 双击启动：`run_ui_prototype.bat`
- 作用：
  - 先看第一版桌面软件的 UI 布局
  - 包含：
    - 总览
    - 账号管理
    - 素材配置
    - 定时规则
    - 任务预览
    - 日志中心
    - 设置
- 说明：
  - 这是**界面原型版**
  - 先看视觉、布局、交互层级
  - 还没接真实业务逻辑

## 2. 旧实验文件
- `scheduler.py`
- `desktop_scheduler_gui.py`

---

## 安装依赖

```bash
pip install -r requirements.txt
```

---

## 启动第一版 UI 原型

### 命令行
```bash
python ui_prototype_v1.py
```

### 双击运行
```bash
run_ui_prototype.bat
```

## 这个版本的目标
当前重点不是执行能力，而是先把这些做得像真正软件：

- 深色风格
- 左侧导航
- 卡片式总览
- 账号列表 + 详情面板
- 素材池编辑区
- 定时规则配置面板
- 任务预览表格
- 日志页
- 设置页
