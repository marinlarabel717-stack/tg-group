部署说明

先用临时地址预览，后面再换成你的正式域名

这套官网是纯静态站，后面部署和换域名都很轻。

部署方式

  • 直接部署 `website/` 目录
  • 可用 Vercel / Netlify / Cloudflare Pages
  • 先用平台给的临时域名预览

后续替换

  • 把联系按钮替换成你的 Telegram / 客服方式
  • 把下载按钮替换成安装包地址
  • 把正式域名解析到部署平台
